# Kitchen Display System (KDS) Документация

## Что это?

Kitchen Display System (KDS) - это экран на кухне, который показывает заказы в реальном времени и помогает команде кухни эффективно работать.

## Основные функции KDS

### 1. Отображение заказов
- Показывает новые заказы в очередности
- Цветные индикаторы приоритета
- Таймер приготовления каждого блюда

### 2. Управление статусом
- Бармен может отметить блюдо как "готовится"
- Повар может отметить как "готово"
- Официант видит, когда блюдо готово к подаче

### 3. Аналитика
- Время приготовления каждого блюда
- Среднее время в очереди
- Производительность повара

## API для KDS

```
GET /api/orders/pending - Получить активные заказы
PUT /api/orders/:id/status - Обновить статус
POST /api/orders/:orderId/items/:itemId/status - Обновить статус блюда
```

## Развертывание KDS

KDS использует тот же фронтенд React приложение, но с отдельным компонентом KitchenDisplay.tsx, оптимизированным для отображения на кухне.
